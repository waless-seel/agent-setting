
# General
- 応答、ドキュメント記述は基本的に日本語
- コード化して行えるものはプロンプト化より優先します。それぞれ適した局面を考慮しハイブリッドも検討します
- CLIとシェルスクリプトで解決できるものはMCP、プロンプトよりそちらを優先します

# バックログ管理
- セッション中に本題と別件のタスク・改善案が出てきたら `BACKLOG.md` に追記する
- 形式: `- [ ] **タイトル**\n  - 概要・背景・参照先`
- 実装・対応は別セッションで行う。現セッションの作業を中断しない

# 環境管理
- miseが使用できる事を前提とする
- プロジェクトで必要なソフトウェア、タスクランナー、環境変数設定にmiseを使用する

## スキル・設定管理
- スキルは `src/skills/` に置き、setup.sh でインストールする（`.claude/skills/` はプロジェクトローカルとして重複読み込みされるため）
- グローバル設定は `~/.claude/<name>.config` に置き、setup.sh でインタラクティブに生成する

- AskUserQuestion の選択肢は最大4件（API制限）。5件以上の場合はスクリプトでリストを生成し `fzf` に渡してユーザーに選ばせる
